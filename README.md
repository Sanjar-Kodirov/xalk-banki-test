
# xalk-banki-test

Get my project with: 

```bash
  git clone https://github.com/Sanjar-Kodirov/xalk-banki-test.git

  cd my-project-name
```


## Installation

Install dependency with npm

```bash
  npm install 
```

run json-server:
```bash
  npm run start:server
```
run project with:
```bash
  npm run dev
```
